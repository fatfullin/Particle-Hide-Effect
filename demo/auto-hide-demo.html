Particle Effect Auto-Hide Demo

Нажмите на любой блок, чтобы активировать эффект частиц и увидеть автоскрытие содержимого.

Библиотека: @fatmax/particle-hide-effect

Стандартное автоскрытие
Используется стандартное автоскрытие с плавным исчезанием содержимого через opacity и visibility.
Медленное затухание
Используется медленное затухание (1 секунда) для плавного исчезновения с ease-in-out эффектом.
Пользовательский CSS класс
Использует пользовательский CSS класс для управления скрытием, позволяя полную настройку анимации.

Код для каждого примера:

Стандартное автоскрытие:

// npm install @fatmax/particle-hide-effect import ParticleEffect from '@fatmax/particle-hide-effect'; const effect1 = new ParticleEffect(document.getElementById('demo1'), { autoHideTarget: true, // включено по умолчанию particleDensity: 250 });

Медленное затухание:

const effect2 = new ParticleEffect(document.getElementById('demo2'), { autoHideTarget: true, targetFadeDuration: 1.0, // 1 секунда вместо стандартных 0.3 targetTimingFunction: 'ease-in-out', particleDensity: 250 });

Пользовательский CSS класс:

const effect3 = new ParticleEffect(document.getElementById('demo3'), { autoHideTarget: true, targetHideClass: 'custom-hide-class', // используем свой CSS класс particleDensity: 250 });
